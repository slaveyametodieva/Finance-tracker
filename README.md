# Personal Finance Tracker (C++)

** Personal Finance Tracker** е конзолно приложение за управление на личния месечен бюджет. То позволява лесно въвеждане на приходи и разходи, следене на баланса, сортиране на месеците по финансови показатели и прогнозиране на бъдещи спестявания.

## Функционалности

Приложението поддържа следните команди, които се въвеждат в конзолата:

| Команда | Описание |
| :--- | :--- |
| **`setup`** | **Задължителна първа стъпка.** Задава броя на месеците (1-12) и заделя необходимата памет. |
| **`add`** | Добавя данни за конкретен месец (приход и разход). Автоматично изчислява баланса. |
| **`report`** | Извежда подробна таблица с всички въведени месеци, както и обобщена статистика (Общо приходи/разходи, Среден баланс). |
| **`search`** | Търси информация за конкретен месец по име (напр. `search January`). |
| **`sort`** | Класира "Топ 3" месеца по Приход, Разход или Баланс, без да нарушава хронологичния ред на данните. |
| **`forecast`** | Изчислява прогнозни спестявания след N месеца на базата на текущата средна месечна промяна. |
| **`chart`** | Визуализира финансовото състояние графично чрез ASCII стълбове. |
| **`exit`** | Освобождава динамичната памет и затваря приложението. |

## Техническа реализация

Проектът демонстрира ключови концепции в C++ програмирането:
* **Динамична памет (Dynamic Memory):** Използване на `new` и `delete` за създаване на масива от месеци по време на изпълнение.
* **Структури (Structs):** Организиране на данните в потребителски тип `Month`.
* **Алгоритми:** Сортиране (Bubble Sort с помощен масив), търсене и математически прогнози.
* **Валидация на входа:** Защита от въвеждане на грешни типове данни (напр. букви вместо цифри) чрез `cin.fail()`.
* **Enumerations & Switch:** Използване на `enum` за по-чиста логика при обработката на команди.
